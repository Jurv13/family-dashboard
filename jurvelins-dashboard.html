🔑 Connect Google Tasks & Calendar

To load live tasks and calendar data you need a Google OAuth 2.0 Client ID. One-time setup (~5 min). Because OAuth requires an HTTP origin (not file://), serve this file via a local web server using Step 5.

1
Go to console.cloud.google.com → create a new project (e.g. "Family Dashboard").
2
APIs & Services → Library: enable Google Tasks API and Google Calendar API.
3
Credentials → Create Credentials → OAuth 2.0 Client ID → type: Web application.
4
Under Authorized JavaScript origins add: http://localhost:8080
5
Open Terminal, navigate to this file's folder, run:
python3 -m http.server 8080
Then open http://localhost:8080/jurvelins-dashboard.html in Chrome.
